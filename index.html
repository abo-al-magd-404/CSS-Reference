Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. It controls layout, colors, fonts, and overall visual appearance.

HTML Example:

<h1 class="title">Welcome to CSS</h1>
<p class="description">This is a sample paragraph</p>

CSS Example:

.title {
    color: var(--color-highlight);
    font-size: 2.5rem;
    text-shadow: 0 2px 10px rgba(255, 204, 0, 0.3);
}

.description {
    color: #e0e0e0;
    font-size: 1.1rem;
    line-height: 1.6;
}

Grouping and Combining Selectors

CSS allows you to group multiple selectors together and combine them to create more specific targeting rules.

HTML Example:

<div class="container">
    <h2>Main Title</h2>
    <p>Paragraph text</p>
    <span>Inline text</span>
</div>

CSS Example:

/* Grouping selectors */
h1, h2, h3 {
    color: var(--color-highlight);
    font-weight: bold;
}

/* Descendant selector */
.container p {
    color: var(--color-text);
    margin-bottom: 10px;
}

/* Child selector */
.container > span {
    color: var(--color-accent);
    font-weight: bold;
}

Result:

Main Title

Paragraph text

Inline text
